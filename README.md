# Prettify JSON
Import json string from file and output it to console in pretty format

#Examples

```
$ python3 pprint_json.py 123.txt
```
text in file :  
```{"foo":{"bar":["baz",null,1.0,2]}}```

# Output
```
{
    "foo": {
        "bar": [
            "baz",
            null,
            1.0,
            2
        ]
    }
}
```
